morse-libs
==========
A list of morse code translator programs in various programming languages and paradigms, for fun.

Usage:
------
morse-libs/C++$ make all
morse-libs/C++$ echo "hello, world" | ./main
.... . .-.. .-.. --- --..--   .-- --- .-. .-.. -.. 
HELLO, WORLD
